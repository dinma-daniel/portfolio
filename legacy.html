[05] About

I build things that are
creative and technically precise.

I'm a software developer and UI/UX designer with a strong background in full-stack engineering and data science. Across the last six years I've built blockchain dashboards, real-time internal analytics, hackathon-winning prototypes, and earned an MSc in Data Science along the way.

I'm fluent in modern JavaScript frameworks — React, Vue, Next.js, Node — and the design tooling that surrounds them. I care as much about the line weight of a border as the name of a database column.

Right now I'm building dashboards for Cyrus Terminal, a Solana pre-bond listings platform — while taking on a small number of independent briefs per quarter where engineering, data and design need to be thought about at the same time, by the same person.
